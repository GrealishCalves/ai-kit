## tools

- Never use `jq`, always use `jaq` for JSON processing (faster, same syntax)
- Never use `grep`, always use `rg` (ripgrep) for text patterns
- Never use `find`, always use `rg --files` for file listing
- Never use `cat`, use `bat` for file viewing with syntax highlighting
- Prefer `fd` over `find` if ripgrep not available
- Never use `cat` for viewing files, always use `bat`
